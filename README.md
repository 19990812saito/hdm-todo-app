# HDM ToDo App

目標から逆算して、担当・期限・進捗・成果ログを管理するHDM向けToDoアプリです。

## Version

- v0.5.0 / 2026-06-01

## 初期機能

- 月間タイムライン
- 担当者別タスク表示
- タスク追加・編集
- 進捗更新
- 成果ログ
- 添付ファイル
- AI管理コメント
- PWA対応
- GitHub Pages公開
- Supabaseクラウド保存
- PCブラウザ通知
- モバイル向け現場入力
- 簡易権限モード

- 権限管理の本実装
- Slack / LINE連携
