# HDM ToDoアプリ スライド用概要

対象: 那覇空港店メンバー向け  
目的: HDM ToDoアプリを使う理由と、基本の考え方を短く伝える

## アプリ名

HDM ToDoアプリ

## 一言でいうと

月間目標から逆算して、誰が・何を・いつまでに・どこまで進めているかを見える化する業務進捗管理アプリです。

## なぜ作るのか

これまでの課題:

- タスクがカレンダー、チャット、口頭に分かれていた
- 誰が何を担当しているか見えにくかった
- 作業が進んでいるのか、止まっているのか分かりにくかった
- 完了した作業や相談内容がログとして残りにくかった
- 月間目標と日々の作業がつながりにくかった

このアプリで解決したいこと:

- 今月やることを全員で共有する
- 担当者ごとの仕事を見える化する
- 期限と進捗を確認しやすくする
- 作業ログを残して、成果として見えるようにする
- 止まっている作業を早めに相談できるようにする

## 基本の考え方

目標があるから、タスクが生まれる。  
タスクには担当と期限がある。  
作業したらログを残す。  
完了したら成果として残る。

## 6月の大きな目的

7月以降に向けて、車両管理・環境整備・安全対策の土台を6月中に整える。

## 6月の重点項目

1. 車両傷チェックアプリの初期登録と本格運用
2. ヤード・トレーラー環境整備
3. ガラス撥水コーティング
4. LINE管理アプリ作成
5. 会計・車検点検・シフトなど月末業務の管理

## 使い方の基本

1. アプリを開く
2. 自分の名前を選ぶ
3. 担当タスクを見る
4. 作業したらログを残す
5. 進捗を更新する
6. 完了したら完了登録する
7. 必要なら写真や資料を添付する

## このアプリで大事にしたいこと

- 作業をやりっぱなしにしない
- 困ったら早めに相談する
- 完了したことを成果として残す
- 店舗全体で目標に向かって進める
- 7月以降の忙しい時期に困らない状態を作る

## スライドで伝えたいメッセージ

HDM ToDoアプリは、ただのToDoリストではありません。  
月間目標を実行に移し、担当・期限・進捗・成果ログを残すための、那覇空港店の業務管理の土台です。
